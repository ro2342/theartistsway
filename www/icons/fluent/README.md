# Ícones (Fluent System Icons)

Subconjunto curado de `microsoft/fluentui-system-icons` (MIT), baixados
como SVG estático — sem depender de nenhum pacote JS pra isso.

| Arquivo | Ícone original | Uso |
|---|---|---|
| `home-regular.svg` / `home-filled.svg` | Home | Nav: Início |
| `book-regular.svg` / `book-filled.svg` | Book | Nav: Jornada |
| `heart-regular.svg` / `heart-filled.svg` | Heart | Nav: Artist Date |
| `settings-regular.svg` / `settings-filled.svg` | Settings | Nav: Ajustes |
| `calendar.svg` | Calendar | Lembretes de calendário |
| `dice.svg` | Arrow Shuffle | Sortear ideia |
| `checkmark-circle.svg` | Checkmark Circle (filled) | Feito |
| `pin.svg` | Location | Regras da Estrada |
| `star.svg` | Star | Princípios Básicos |
| `bell.svg` | Alert | Notificações |
| `arrow-left.svg` | Arrow Left | Voltar |
| `person.svg` | Person | Nome/perfil |
| `sync.svg` | Arrow Sync | Sincronização |
| `tools-regular.svg` / `tools-filled.svg` | Wrench | Nav: Ferramentas |
| `delete.svg` | Delete | Zona de risco |
| `edit.svg` | Edit | Editar |
| `save.svg` | Save | Salvar |
| `chevron-right.svg` | Chevron Right | Navegação/expandir |
| `moon.svg` | Weather Moon | Tema escuro |
| `sun.svg` | Brightness High | Tema claro |
| `color.svg` | Color | Accent color |
| `warning.svg` | Warning | Avisos |

Todos em 20px (24px pros 4 de navegação), estilo `regular` (ou `filled`
onde marcado). Pra adicionar mais, ver a estrutura em
`assets/<Nome>/SVG/ic_fluent_<slug>_<tamanho>_<estilo>.svg` no repositório
oficial.
